Colors can be represented in :

  • Color Keyword :
    • We simply use color keyword like - blue
  • Hex color code :
    • We use # code values of Color... for blue - #0000ff
  • RGB :
    • We use r,g,b value of color.. for purple - rgb(128, 0, 128)
  • RGBA :
    • Same as rgb but a refers to opacity of color how dark or light we want that color
  • HSL :
    • hsl = hue(h), saturation(s), lightness(l)
